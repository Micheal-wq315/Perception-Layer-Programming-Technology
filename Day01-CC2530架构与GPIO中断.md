## 二、CC2530 四个内存空间（核心考点）

| 空间 | 总线 | 大小 | 用途 | 访问 |
|------|------|------|------|------|
| **CODE** | 16位 | 64KB | 程序代码、常量 | 只读 |
| **DATA** | 8位 | 256B | 运行变量 | 读写，低128直接/高128间接 |
| **XDATA** | 16位 | 64KB | DMA 访问、大容量数据 | 读写，较慢 |
| **SFR** | — | 128B | 控制 CPU/外设寄存器 | 读写，部分可位寻址 |

### 2.1 物理存储器 vs 存储空间

- **存储空间** = 寻址方式概念（CODE/DATA/XDATA/SFR）
- **物理存储器** = 实际硬件（Flash、SRAM、寄存器）
- 两者通过**映射**关联，方便 DMA 访问所有物理存储

### 2.2 Flash 控制器

| 特性 | 值 |
|------|-----|
| 页大小 | **2 KB** |
| 擦除最小单位 | 页 |
| 写入最小单位 | **4 字节** |
| 擦写寿命 | ~20,000 次 |
| 信息页面 | 2KB，只读，存 IEEE 地址 |

### 2.3 Flash 写入六步算法

```
1. 设置 FADDRH:FADDRL = 起始地址（18位地址的高16位）
2. 设置 FCTL.WRITE = 1，启动写入状态机
3. 在 20μs 内向 FWDATA 写 4 次（LSB 先写）
4. 等待 FCTL.FULL 变低（Flash 控制器已开始编程）
5. 可选状态检查：超时 → FCTL.BUSY/WRITE=0；页锁定 → FCTL.ABORT=1
6. 若不是最后 4 字节则跳到步骤 3
```

> ⚠️ Flash 写入期间 CPU 不能访问 Flash，写入代码必须从 SRAM 执行！

### 2.4 Bank 映射

- CC2530F256 = 256KB Flash = 8 个 bank × 32KB
- CODE 寻址空间只有 64KB，通过 **FMAP.MAP[2:0]** 选择映射哪个 bank 到 0x8000-0xFFFF
- XDATA 通过 **MEMCTR.XBANK[2:0]** 选择映射

## 三、中断系统

### 3.1 中断配置五步（必背）

```
① 清除中断标志位（清遗留标志）
② 设置外设 SFR 寄存器中的中断使能位
③ 设置 IEN0/IEN1/IEN2 中对应的中断使能位为 1
④ 设置 IEN0.EA = 1（全局中断使能）
⑤ 在对应中断向量地址编写中断服务程序 (ISR)
```

### 3.2 中断核心概念

- **18 个中断源**，分为 **6 个优先级组 (IPG0-IPG5)**
- 每组由 `IP1_x` 和 `IP0_x` 两位编码，共 **4 个优先级级别**
- 正在执行的 ISR **只能被更高优先级中断打断**
- 同优先级同时到达 → 按**固定硬件轮询顺序**响应
- 最快中断响应时间 = **7 个机器周期**（1 周期检测 + 6 周期 LCALL）
- 中断返回用 **RETI** 指令

### 3.3 中断使能寄存器速查

| 寄存器 | 关键位 |
|--------|--------|
| **IEN0** | EA(bit7) 全局使能，URX0IE(bit2) 串口0收，URX1IE(bit7)串口1收 |
| **IEN1** | P0IE(bit5) P0口中断 |
| **IEN2** | P1IE(bit4) P1口中断，P2IE(bit1) P2口中断，UTX0IE/UTX1IE |

### 3.4 中断标志寄存器

| 寄存器 | 关键位 |
|--------|--------|
| **TCON** | URX1IF(bit7)、ADCIF(bit5)、URX0IF(bit3) |
| **IRCON** | STIF(bit7)、T4IF-T1IF(bit4-1)、DMAIF(bit0) |
| **IRCON2** | WDTIF(bit4)、P1IF(bit3)、UTX1IF(bit2)、UTX0IF(bit1)、P2IF(bit0) |
| **P0IFG/P1IFG/P2IFG** | 端口中断标志 |

### 3.5 GPIO 中断配置寄存器

| 寄存器 | 功能 |
|--------|------|
| **PICTL** | 配置 P0/P1/P2 边沿触发（上升沿/下降沿） |
| **P0IEN/P1IEN/P2IEN** | 各引脚中断单独使能 |
| **P0IFG/P1IFG/P2IFG** | 各引脚中断标志 |

## 四、GPIO（通用输入输出）

### 4.1 三寄存器配置口诀

```
PxSEL  → 选功能：0=通用IO, 1=外设功能
PxDIR  → 选方向：0=输入, 1=输出
PxINP  → 配上下拉：0=上拉/下拉, 1=高阻(三态)
```

### 4.2 常用配置组合

| 目标配置 | PxSEL | PxDIR | PxINP | Px |
|----------|-------|-------|-------|----|
| 推挽输出高 | 0 | 1 | — | 1 |
| 推挽输出低 | 0 | 1 | — | 0 |
| 上拉输入 | 0 | 0 | 0 | — |
| 下拉输入 | 0 | 0 | 1（配合P2INP） | — |
| 高阻输入 | 0 | 0 | 1 | — |
| 外设功能 | 1 | — | — | — |

### 4.3 输出模式

| 模式 | 特点 |
|------|------|
| **推挽输出** | 可主动输出高/低电平，驱动能力强 |
| **开漏输出** | 只能输出低或高阻，需外接上拉 |
